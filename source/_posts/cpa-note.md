## String methods

### compare

str1.compare(str2)

- str1.compare(str2) == 0 when str1 == str2

- str1.compare(str2) > 0 when str1 > str2

- str1.compare(str2) < 0 when str1 < str2

S.compare(substr_start, substr_length, other_string)

S.compare(substr_start, substr_length, other_string, other_substr_start, other_substr_length)
